     This is a site I created to show off my skills at HTML and Javascript. I have made an incredible amount of progress in both of these languages since 6th grade. The entire purpose of this project is to show somewhere you have grown and with this being my top subject I felt it to be very appropriate to make a from scratch website to showcase it.

     Back in 6th grade I thought I ruled the IT world knowing nothing more than how to power off and on again. Now I can create apps, Javascript programs, websites and more. Now you might be saying, “What about 7th grade, what did you know then?” Back in 7th grade I knew just a little more than I did in 6th grade but that is when I really found my love for coding all manner of things. At the end of the year I knew that I wanted to be in the first ever PIT (Principles of Information Technology) class so I put that on my requested schedule and I got in.

     This class has helped me realize my potential and teach me skill I might need later down the road of software development/engineering. Even this project coerced* me into learning. I had to learn bootstrap to be able to complete this project effectively.

I hope you enjoy my creation!
